# Angular2 NPM Starter
> by Fusion Alliance - @fusionalliance

Angular2 project using NPM scripts as the build tool.

## Getting Started

### To run:

Install dependencies:
```shell
npm install
```

Then after the dependencies have been installed:

```shell
npm start
```
